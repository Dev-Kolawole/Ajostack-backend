THE AJOSTACK MASTER BLUEPRINT
1. Project Title & One-Sentence Pitch
AjoStack — An automated dues collection platform that helps Cooperative Society treasurers collect and reconcile monthly contributions at scale.

2. The Problem
Every month, a Cooperative Society treasurer sends the same WhatsApp broadcast, makes the same phone calls, and still ends up with an incomplete collection. They waste hours manually confirming bank receipts and updating a reconciliation spreadsheet that is perpetually three versions behind. The society loses money to delayed contributions, and trust erodes when manual records don't match.

3. The Solution
We eliminate the manual collection bottleneck. Members securely tokenize their cards at onboarding, allowing scheduled contributions to be processed automatically via the Nomba API. Instead of chasing 50 different bank transfers, the treasurer gets a real-time dashboard showing successful collections, failed charges (NSF/Expired), and monthly totals, drastically reducing administrative friction.
